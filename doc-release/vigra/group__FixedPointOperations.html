TARGET vigra::fixed_point_cast ( FixedPoint< IntBits, FracBits >  v)

Convert a FixedPoint to a built-in type. If the target is integral, the value is rounded.
Usage:

FixedPoint<16,15> fp(...);
double d = fixed_point_cast<double>(fp);
