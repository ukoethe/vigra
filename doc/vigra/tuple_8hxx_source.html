00043 /** \page TupleTypes Tuple Types
00044 
00045     pair, triple, tuple4, tuple5
00046 
00047     <b>\#include</b> <vigra/utilities.hxx><br>
00048     Namespace: vigra
00049 
00050     VIGRA defines tuple types \p vigra::triple, \p vigra::tuple4, \p vigra::tuple5.
00051     In addition, \p std::pair is imported into namespace vigra from the C++ standard
00052     library. All these types are defined similarly:
00053 
00054     <ul>
00055 
00056     <li> They are parameterized by the respective number of types. For each tuple,
00057     a constructor is defined that takes that many arguments, e.g.:
00058     \code
00059     template <class First, class Second, class Third>
00060     class Triple { ... };
00061     \endcode
00062     </li>
00063     <li> A number of \p typedef's tells the types stored in the tuple:
00064 
00065     \code
00066     typedef ... first_type;
00067     typedef ... second_type;
00068     typedef ... third_type;  // triple, tuple4, tuple5 only
00069     typedef ... forth_type;  // tuple4, tuple5 only
00070     typedef ... fifth_type;  // tuple5 only
00071     \endcode
00072     </li>
00073     <li> Items are stored in the following public attributes:
00074 
00075     \code
00076 
00077     first;
00078     second;
00079     third;  // triple, tuple4, tuple5 only
00080     forth;  // tuple4, tuple5 only
00081     fifth;  // tuple5 only
00082 
00083     \endcode
00084     </li>
00085     </ul>
00086 
00087 
00088 */
